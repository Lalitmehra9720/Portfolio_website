# 🌟 Lalit Mehra - Portfolio Website

A modern, interactive portfolio website showcasing my work as a Full-Stack Developer. Built with a focus on performance, clean UI, and smooth animations.

---

## 🚀 Live Demo

🔗 https://portfolio-2tkm.vercel.app/

---

## ✨ Features

- ⚡ Interactive Hero Section with animations & particle background  
- 🎯 Clean About Section with stats & skills overview  
- 🛠️ Skills Section with categorized tech stack  
- 💼 Projects Showcase with live demo & GitHub links  
- 📩 Contact Form with email integration  
- 🌗 Dark/Light Mode toggle  
- 🎨 Smooth animations using Framer Motion  
- 📱 Fully responsive design  

---

## 🚦 Getting Started

### 1️⃣ Clone the repo

```bash
git clone https://github.com/Lalitmehra9720/Portfolio.git
cd your-portfolio
```

### 2️⃣ Install dependencies

```bash
npm install
```

### 3️⃣ Run project

```bash
npm run dev
```
